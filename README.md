## 📖 项目简介 (Introduction)

**本项目不是一个拦截软件，而是一份对抗算法的“操作手册”。**

在当今的互联网环境下，我们享受便利的同时，也成为了大数据的“透明人”。骚扰电话、精准广告、杀熟机制、算法茧房无处不在。本项目致力于个人隐私保护，基于**“反向大数据”**逻辑，汇总了主流 APP 和系统中隐藏极深的**隐私设置、个性化推荐关闭入口及广告屏蔽方法**。

本质上，这是一个教你如何夺回数字主权的开源教程 @Wangry。

## 🎯 核心理念：什么是“反向大数据”？

通常的“大数据”是收集用户行为来训练算法；而**“反向大数据”**则是通过主动阻断数据采集源头，向算法输入“噪音”或“空值”，从而让算法画像失效。

- **阻断 (Block):** 关闭不必要的权限（麦克风、通讯录、剪切板）。
- **混淆 (Obfuscate):** 拒绝个性化推荐，重置广告 ID。
- **净化 (Purify):** 移除不必要的各类“监听”开关 @Wangry。
