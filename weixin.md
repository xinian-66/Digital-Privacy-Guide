## 🚨 1. 核心隐私开关 (Core Privacy)
*关闭 微信 [**Wechat**] 的这些开关，可以阻断大部分数据收集。*

> [!WARNING]
> **注意**
> 版本更新后设置可能失效，建议定期复查。

### 🔥 关闭个性化推荐
- **路径**: `我的 -> 设置 -> 隐私/个人信息与权限 -> 授权管理`
- **说明**: 查看通过微信已授权的应用[**通过微信登陆的应用**] 可以操作解绑[解除授权]。
- **常见示例**: 王者荣耀、全民K哥、腾讯视频等。

### 👍 广告个性化 / 程序化广告
- **路径**: `我的 -> 设置 -> 隐私/个人信息与权限 -> 个性化广告管理`

| 广告推荐模式 | 状态 | 说明 |
| :--- | :---: | :--- |
| **个性化 [默认]** | × | |
| **随机模式 [用户]** | **√** | [推荐使用随机模式，减少广告] |

- **路径**: `我的 -> 设置 -> 隐私/个人信息与权限 -> 个人信息浏览与导出`
- **说明**: 可以查看包含头像、昵称、微信号、性别、地区、手机号码、QQ号码、邮箱、历史设备等信息。[需要时可导出个人信息]

## 🧭 发现页

### 🎵 听一听
- **路径**: `我的 -> 设置 -> 通用 -> 听一听` (如未使用，建议全部关闭)

* **个性化推荐**: 建议 `关闭`。
* **优先使用无损音质播放**: 建议 `关闭`。
* **在最近中显示该功能**: 建议 `关闭`。

### 👁️ 发现页管理
- **路径**: `我的 -> 设置 -> 通用 -> 发现页管理` (建议关闭未使用的功能)

* **朋友圈**: 建议 `保留`。
* **视频号**: 建议 `关闭`。
* **直播**: 建议 `关闭`。
* **扫一扫**: 建议 `保留`。
* **听一听**: 建议 `关闭`。
* **看一看**: 建议 `关闭`。
* **搜一搜**: 建议 `关闭`。
* **附近的人**: 建议 `关闭`。
* **游戏**: 建议 `关闭`。
* **小程序**: 建议 `有新内容提醒我，建议关闭`。

## ⚙️ 2. 其他设置 (Others)

### 🛠️ 功能管理
- **路径**: `我的 -> 设置 -> 通用 -> 辅助功能` (未使用的功能，建议关闭)

* **QQ邮箱提醒**: 建议 `按需启用`。
* **群发助手**: 建议 `按需启用`。
* **微信支付**: 建议 `启用`。
* **微信收款助手**: 建议 `按需启用`。
* **微信商家支付助手**: 建议 `按需启用`。
* **小程序助手**: 建议 `按需启用`。
* **Facebook辅助功能**: 建议 `按需启用/建议关闭`。
* **腾讯新闻**: 建议 `按需启用/建议关闭`。
* **微信运动**: 建议 `按需启用/建议关闭`。
* **微信游戏**: 建议 `按需启用/建议关闭`。

### 💳 卡包
- **路径**: `我的 -> 卡包` (建议删除不再使用的 证件/卡！)
- **示例**: 点击 `会员卡` -> 点击 `XX会员卡` 右上角 `...` 可查看/删除 会员卡

## 🛡️ 3. 系统权限管理 (Permissions)

> [!TIP]
> **建议**
> 仅授予最小必要权限。

* **位置**: 建议设置为 `使用期间` 或 `永不`。
* **通讯录**: 建议 `禁止` (防杀熟/防关联推荐)。
* **麦克风**: 建议 `使用期间`。
* **摄像头/相机**: 建议 `使用期间`。
* **蓝牙**: 建议 `使用期间`。
* **照片**: 建议 `使用期间`。
